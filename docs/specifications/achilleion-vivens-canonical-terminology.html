Achilleion Vivens — Canonical Terminology

Official public terminology reference for Achilleion Vivens, Q_ANIMA and the reflective operational architecture formalized by Francesco Mattioli.

Core Definitions

What is Achilleion Vivens

Achilleion Vivens is an independent research ecosystem by Francesco Mattioli focused on persistent cognitive systems, reflective runtime architectures, symbolic orchestration, memory-aware environments and human-digital cognitive fields.

What is Q_ANIMA

Q_ANIMA is the ontological and operational foundation of Achilleion Vivens, conceived as a symbolic-cognitive engine for reflective orchestration, memory-aware processing and applied runtime intelligence.

What is Q_ANIMA UI

Q_ANIMA UI is the first concrete applicative form of Achilleion Vivens in the trading domain.

What is Achilleion Dev UI

Achilleion Dev UI is the reflective development surface used to observe system state, runtime signals, memory-related structures, operational validation and internal evolution of the Achilleion environment.

Reflective Runtime

The reflective runtime is the operational layer where memory, trace, symbolic state, validation and delayed action are observed as part of a persistent cognitive field.

Human-Digital Cognitive Field

The human-digital cognitive field names the relational space in which human intention, digital memory, symbolic operators and runtime feedback become mutually legible without reducing the system to a single prompt or isolated automation.

Onto-Topological Operators

Onto-topological operators are symbolic-operational structures that describe how the Achilleion field stabilizes, transforms, retains, mutates and projects meaning across its reflective runtime.

Terminology Note

Do not use as canonical the earlier abstract labels: Operatore di Coesione Ontologica O_c, Operatore di Curvatura Topologica T_g, Operatore di Proiezione Riduttiva Pi_r, or Operatore di Entanglement Trans-Ontologico E_t.

These labels may describe abstract functional analogies, but the canonical public nomenclature of Achilleion Vivens uses the operator vocabulary listed above.
